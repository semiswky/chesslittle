# chesslittle

**chesslittle**: React tabanlı dokunmatik uyumlu satranç oyunu.  
- İnsan vs Bot  
- Rok, terfi ve en passant destekli  
- Mobil uyumlu ve akıcı animasyonlar  
- Her oyuncunun 10 dakikalık süresi  

## Kullanım

1. Taşa dokun → sonra hedef kareye dokun  
2. "Yeni oyun" butonuna basarak oyunu başlat

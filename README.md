# ChessLittle

React tabanlı dokunmatik uyumlu satranç oyunu: İnsan vs Bot, rok/terfi/en passant destekli, mobil uyumlu.

## Özellikler
- İnsan vs Bot oynanabilir.
- Rok, terfi ve en passant hamleleri desteklenir.
- Her oyuncu 10 dakika süreye sahiptir.
- Mobil cihazlar için dokunmatik uyumludur.
- Bot sınırlı süre içinde en iyi hamleyi arar.

## Kullanım
1. Oyun başlatmak için “Yeni oyun” butonuna tıklayın.
2. Önce taşınıza, sonra hedef kareye dokunun.
3. Oyunu sıfırlamak için “Yeni oyun” tuşunu kullanın.
